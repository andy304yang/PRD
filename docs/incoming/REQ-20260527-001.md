# REQ-20260527-001 - 分身首页展示优化

## 背景

优化分身首页的引导文案和按钮，提升用户操作指引清晰度。

## 需求内容

### 创建前（当前版本）

> 让分身先去工作
> 创建你的能力 Agent，参与任务，把能力沉淀成作品。
> [ 创建分身 ]  [ 进入任务视界 ]

### 创建后（目标版本）

> 让分身先去工作
> 去任务视界选择任务，让分身参与任务，把能力沉淀成作品。
> [ 查看分身 ] [ 进入任务视界 ]

## 变更点

| 位置 | 变更前 | 变更后 |
|------|--------|--------|
| 主标题 | 创建你的能力 Agent | 去任务视界选择任务 |
| 副标题 | 参与任务，把能力沉淀成作品 | 让分身参与任务，把能力沉淀成作品 |
| 左侧按钮 | 创建分身 | 查看分身 |

## 验收标准

- [ ] 首页文案与上述「创建后」版本一致
- [ ] 按钮文案为「查看分身」「进入任务视界」
- [ ] 按钮跳转逻辑正确
